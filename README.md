# Crop Meme Generator

### Installation
```
$ git clone https://github.com/Allespro/CropMeme
$ cd CropMeme
$ pip3 install -r requirements.txt
# PUT YOUR IMAGE IN CropMeme FOLDER
$ python3 main.py
```
